# Moment Capsule — 全版本功能清单

> 产品定位：事件驱动的情感时间胶囊，不是日记 App
> 最后更新：2026-03-20

---

## v0.1 — 可用原型（最小闭环）

**版本目标**：跑通"封存 → 地图 → 回看"核心链路，尽快开始自用
**平台**：微信小程序 + 微信云开发
**预估周期**：2 周

| # | 功能 | 功能描述 | 工作量 |
|---|------|----------|--------|
| 0.1 | 微信登录 | 静默登录，获取 openId，无需额外注册流程 | S |
| 0.2 | 云开发存储 | 云数据库（胶囊数据）+ 云存储（媒体文件）+ 云函数 | M |
| 0.3 | 地理位置自动捕获 | 打开封存页自动获取经纬度 | S |
| 0.4 | 地点名称反查 | 经纬度 → 可读地名（腾讯地图逆地理编码），封存时自动填充，回看时显示 | S |
| 0.5 | 文字记录 | 多行文本输入，无字数限制，支持换行 | S |
| 0.6 | 拍照/选图 | 支持现场拍照或从相册选取，最多 9 张。位置 = 封存时所在位置，非照片 EXIF | S |
| 0.7 | 封存保存 | 将文字+照片+位置+地名+时间打包为一颗胶囊，上传云端 | S |
| 0.8 | 地图视图 | 首页为地图，胶囊以标记散落在地图上，标记显示地名，可缩放浏览 | M |
| 0.9 | 沉浸式回看（基础版） | 点击胶囊进入详情：照片全屏铺底、文字淡入叠加、地名+时间显示 | L |

**v0.1 验证的核心假设：**
1. 从打开到封存完成是否真的 ≤2 分钟？
2. 回看时感受能不能"回来"？
3. 地图散落的胶囊是否让人想点开？

### v0.1 实现顺序

```
Week 1:
├── 0.1 微信登录 + 0.2 云开发存储（数据模型、云函数）
├── 0.3 地理位置 + 0.4 地点名称反查
├── 0.5 文字记录 + 0.6 拍照/选图 + 0.7 封存保存
└── 产出：可以封存胶囊了

Week 2:
├── 0.8 地图视图
├── 0.9 沉浸式回看
└── 产出：可以在地图上看到胶囊并回看了
```

## v1.0 — 微信小程序 MVP（在 v0.1 基础上补齐）

**版本目标**：能封存、能重逢、体验完整
**前提**：v0.1 自用验证核心假设后
**平台**：微信小程序 + 微信云开发
**预估周期**：v0.1 之后再 2-3 周

### P0 — 完整体验必须

| # | 功能 | 功能描述 | 工作量 |
|---|------|----------|--------|
| 1.1 | 照片压缩 | 上传前在小程序端压缩到 1MB 以内（缩至 1080p + quality 0.8） | S |
| 1.2 | 时间线视图 | 按时间倒序展示胶囊列表，卡片含封面照片+文字摘要+地名+时间 | S |
| 1.3 | 时间筛选 | 时间线视图支持按年/月筛选 | S |
| 1.4 | 封存仪式动效 | 保存时有"封存"的视觉反馈（如胶囊封口、飘入地图等），不是简单的"已保存" | M |
| 1.5 | 短视频录制 | 支持录制 ≤15 秒短视频（替代 Live Photo），引导用户拍 3-5 秒氛围片段 | S |
| 1.6 | 环境音录制 | 独立录音按钮，≤60 秒，录制中有波形/时长反馈 | S |
| 1.7 | 沉浸式回看增强 | 在基础版上增加：短视频自动播放、环境音自动播放、多图左右滑动 | M |

### P1 — 重要增强

| # | 功能 | 功能描述 | 工作量 |
|---|------|----------|--------|
| 1.8 | 离线封存 | 无网络时胶囊保存到本地缓存，恢复网络后自动同步到云端，状态可见 | M |
| 1.9 | 胶囊编辑 | 进入详情后可编辑文字内容 | S |
| 1.10 | 胶囊删除 | 长按或详情页内删除，二次确认 | S |
| 1.11 | 首次使用引导 | 引导页说明产品理念（"不是日记，是时间胶囊"），分步授权定位+相机+麦克风 | S |
| 1.12 | 空状态设计 | 无胶囊时显示温暖的等待文案，不施压，不催促 | S |
| 1.13 | 权限拒绝降级 | 定位被拒 → 手动标记位置或标记为"未知位置"；相机被拒 → 仅可选相册；麦克风被拒 → 隐藏录音入口 | S |
| 1.14 | "历史上的今天" | 每次打开小程序时，如果去年/上月今天有胶囊，首页顶部浮现提示 — v1.0 阶段的轻量"被动重逢" | S |

### v1.0 实现顺序（v0.1 之后）

```
Phase A: 封存增强（~1 周）
├── 1.1 照片压缩
├── 1.4 封存仪式动效
├── 1.5 短视频录制        ← 可并行
├── 1.6 环境音录制        ← 可并行
└── 1.8 离线封存

Phase B: 回看增强（~1 周）
├── 1.2 时间线视图
├── 1.3 时间筛选
├── 1.7 沉浸式回看增强
└── 1.14 "历史上的今天"

Phase C: 完善（~0.5-1 周）
├── 1.9 胶囊编辑
├── 1.10 胶囊删除
├── 1.11 首次使用引导
├── 1.12 空状态设计
└── 1.13 权限拒绝降级
```

## 数据模型（v0.1 起，兼容后续版本）

```
Capsule {
  _id:          string          // 胶囊唯一 ID
  openId:       string          // 用户标识
  text:         string          // 文字内容
  photos:       string[]        // 照片云存储路径（最多 9 张）
  video:        string | null   // 短视频云存储路径（v1.0 起，≤15秒）
  audio:        string | null   // 环境音云存储路径（v1.0 起，≤60秒）
  location: {
    latitude:   number
    longitude:  number
    name:       string          // 地点名称（逆地理编码）
    address:    string | null   // 详细地址
  }
  weather:      object | null   // v1.1 天气数据
  createdAt:    timestamp       // 封存时间
  updatedAt:    timestamp       // 最后编辑时间
  syncStatus:   string          // "synced" | "pending" | "failed"
  timeLock:     timestamp | null // v2.0 时间锁到期时间
}
```
