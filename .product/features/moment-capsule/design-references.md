## 一、设计方向总结

基于 40+ 个参考作品的分析，"此刻胶囊"的设计方向应该是：

> **温暖极简主义 + 沉浸式照片体验 + 仪式感交互**

### 核心设计关键词

- **色彩**：日落色系（琥珀、暖橙、柔粉）+ 米白/奶白底色 + 深色沉浸模式（回看时）
- **质感**：柔和渐变、毛玻璃、轻微阴影，不冰冷不尖锐
- **排版**：大留白、大字号、宽行距，像翻开一本手写本
- **交互**：流畅过渡动画、照片优先、内容即界面
- **氛围**：安静、克制、温暖，像一个安静的下午

## 三、5 个核心设计启发

### 1. "平静的日常仪式"
将封存设计为一个**仪式而非任务**。ZIBRI 和 Monnday 都在日常记录中注入了仪式感 — 不是"填一个表单"，而是"完成一个小小的仪式"。

→ **对此刻胶囊的影响**：封存页的视觉要安静、从容，封存完成的动效要有"完成感"而非"提交感"。

### 2. "内容即界面"
tubik 的 Gallery App 和 Photo App 证明了一个原则：**照片本身就是最好的界面**。UI 元素应该退到照片后面，而非与照片竞争视觉注意力。

→ **对此刻胶囊的影响**：回看页应该让照片占据 80%+ 的视觉面积，文字、地名、时间都是"附着在照片上的"，不是独立的信息区块。

### 3. "两种模式的色彩对比"
日常操作（封存、浏览）用**浅色温暖底色**；沉浸式回看用**深色/照片铺底**。这种对比暗示用户"进入了另一个时空" — 正是"重逢"的感觉。

→ **对此刻胶囊的影响**：地图页和时间线用奶白暖色调；点击胶囊进入回看后，整个色调切换为深色沉浸模式。

### 4. "地图标记要有温度"
默认地图标记（红色钉子）没有情感。参考 VIBE MAP 和 Travel App 的做法 — 用自定义的温暖色调小图标，让地图上散落的不是"数据点"而是"回忆"。

→ **对此刻胶囊的影响**：Marker 设计为琥珀色的小胶囊/小瓶子图标，带有微弱的发光效果。

### 5. "时间胶囊的开启感"
Time Capsule 类参考都强调"开启"的惊喜感。即使此刻胶囊不做时间锁，进入回看时也应该有一个短暂的"开启"过渡 — 不是直接跳转，而是像打开一个封好的瓶子。

→ **对此刻胶囊的影响**：从地图 Marker/时间线卡片 → 回看页的转场动画，应该有"展开/开启"的感觉（0.3-0.5s）。

## 五、此刻胶囊的视觉风格定义（草案）

基于以上参考，提炼出的风格定义：

```
风格名称：Warm Capsule（温暖胶囊）

色彩：
  - 日常模式：奶白底(#FFF8F0) + 琥珀橙(#F5A623) + 暖灰文字(#4A4A4A)
  - 沉浸模式：深炭灰底(#2C2C2E) + 白色文字 + 照片铺底
  - 点缀色：柔粉橙(#E8836B) + 暖杏(#FFD9A0)

字体：
  - 中文：系统默认（PingFang SC），偏大字号（16-18pt 正文）
  - 英文/数字：SF Pro Rounded 或类似圆角字体
  - 行间距：1.6-1.8 倍

圆角：
  - 卡片：12-16px
  - 按钮：全圆角
  - 照片：8-12px

阴影：
  - 柔和扩散阴影，不要硬边阴影
  - 示例：0 4px 16px rgba(0,0,0,0.06)

动效原则：
  - 缓入缓出（ease-in-out），不要线性
  - 时长 0.3-0.5s，不要太快也不要拖沓
  - 淡入淡出 > 滑入滑出
  - 所有过渡都要有动画，不允许硬切
```
