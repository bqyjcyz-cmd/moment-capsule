# Moment Capsule — 交互操作规格（完整版）

> 定义每个页面的交互细节、按钮行为、状态流转、转场动画和异常处理
> 最后更新：2026-03-22

---

## 0. 全局用户旅程

```
首次打开
  ↓
引导页 1/3（产品理念）→ 引导页 2/3（不是日记）→ 引导页 3/3（权限授权）
  ↓
地图首页（空状态）
  ↓
点击"封存"按钮
  ↓
封存页（自动定位 → 写字 → 拍照/视频/环境音 → 封存）
  ↓
封存仪式动效 → 自动跳转回地图首页（新胶囊 Marker 落下）
  ↓
日常使用：地图浏览 / 时间线浏览 → 点击胶囊 → 沉浸式回看
```

- 仅展示一次，完成后写入 `hasOnboarded = true`
- 后续打开直接进入地图首页

### 1.4 引导页 3/3 — 权限授权

```
┌─────────────────────────────────┐
│                                 │
│          (暖色渐变背景)           │
│                                 │
│           [定位图标]              │  ← lucide "map-pin"
│                                 │
│     为了记住你的位置和此刻         │
│       需要获取一些权限            │
│                                 │
│   ┌─────────────────────────┐   │
│   │ 📍 位置权限              │   │  ← 权限卡片1
│   │    记住你封存此刻的地方   │   │
│   │                   [授权] │   │  ← 右侧小按钮
│   └─────────────────────────┘   │
│                                 │
│   ┌─────────────────────────┐   │
│   │ 📷 相机权限              │   │  ← 权限卡片2
│   │    拍下此刻的画面        │   │
│   │                   [授权] │   │
│   └─────────────────────────┘   │
│                                 │
│   ┌─────────────────────────┐   │
│   │ 🎙 麦克风权限            │   │  ← 权限卡片3
│   │    录下此刻的声音        │   │
│   │                   [授权] │   │
│   └─────────────────────────┘   │
│                                 │
│       [ 进入此刻胶囊 ]          │  ← 主按钮：始终可点击
│                                 │
│     暂时跳过，之后也可以授权      │  ← 底部灰色小字
│                                 │
│         · · ·                   │  ← 页面指示器：第3个高亮
└─────────────────────────────────┘
```

**按钮行为：**
| 元素 | 操作 | 结果 |
|------|------|------|
| 位置权限"授权"按钮 | 点击 | 调用 `wx.authorize({ scope: 'scope.userLocation' })`，弹出系统授权弹窗 |
| 位置权限 — 系统弹窗"允许" | 点击 | 按钮变为 ✓ 已授权（琥珀色对勾），不可再点击 |
| 位置权限 — 系统弹窗"拒绝" | 点击 | 按钮变为"去设置"，点击后调用 `wx.openSetting()` 打开系统设置页 |
| 相机权限"授权"按钮 | 点击 | 调用 `wx.authorize({ scope: 'scope.camera' })`，同上逻辑 |
| 麦克风权限"授权"按钮 | 点击 | 调用 `wx.authorize({ scope: 'scope.record' })`，同上逻辑 |
| "进入此刻胶囊"按钮 | 点击 | 关闭引导页，写入 `hasOnboarded = true`，进入地图首页（空状态），过渡动画：整页淡出（0.5s） |
| "暂时跳过"文字 | 点击 | 同上，跳过所有未授权的权限 |
| 页面 | 右滑手势 | 返回引导页 2/3 |

**权限卡片状态：**
| 状态 | 视觉 | 按钮文案 |
|------|------|---------|
| 未授权 | 白色卡片，图标为灰色 | "授权" |
| 已授权 | 白色卡片，图标变为琥珀色，右侧显示 ✓ | "已授权"（不可点击） |
| 被拒绝 | 白色卡片，图标为灰色，底部小字提示 | "去设置" |

**设计原则：**
- 三个权限独立授权，不是一次性全弹
- 用户可以只授权位置，不授权相机和麦克风
- 即使全部跳过，也能进入主页（降级使用）
- 不显示"必须授权才能使用"之类的威胁性文案

## 2. 地图页（首页）

### 2.1 页面加载流程

```
进入地图页
  ↓
检查定位权限
  ├─ 已授权 → 获取当前位置 → 地图中心定位到用户位置
  └─ 未授权 → 地图显示默认位置（中国中心或上次缓存位置）
  ↓
从云端加载胶囊列表
  ├─ 有数据 → 渲染 Marker
  └─ 无数据 → 显示空状态文案
  ↓
检查"历史上的今天"
  ├─ 有匹配 → 顶部浮现提示卡片
  └─ 无匹配 → 不显示
```

### 2.2 地图交互

| 操作 | 手势/触发 | 结果 |
|------|----------|------|
| 缩放地图 | 双指捏合/展开 | 地图缩放，Marker 位置跟随 |
| 拖动地图 | 单指拖动 | 地图平移 |
| 点击空白区域 | 单击 | 如果有已打开的预览卡片，收起卡片（0.2s 下滑淡出） |
| 点击胶囊 Marker | 单击 | 底部弹出预览卡片（0.3s 从底部上滑） |
| 点击预览卡片 | 单击 | 进入沉浸式回看页（共享元素过渡动画：卡片照片放大为全屏，0.4s） |
| 回到当前位置 | 点击右下角定位按钮 | 地图动画平移回用户当前位置（0.5s） |

### 2.3 胶囊 Marker 详细规格

```
┌──────┐
│ ●    │  ← 琥珀色渐变圆形，32px 直径，带发光阴影
│      │
└──┬───┘
   │
 南京东路     ← 地名标签，最多6字，10px，暖灰色
```

- **Marker 图标**：琥珀色渐变圆形（#F5A623 → #E8836B），32px
- **发光效果**：`shadow: 0 2px 8px rgba(245, 166, 35, 0.25)`
- **地名标签**：Marker 正下方，10px，#4A4A4A，白色阴影保证在地图上可读
- **不同大小**：最新的胶囊 Marker 稍大（32px），较早的稍小（24px），视觉上暗示时间层次

### 2.4 预览卡片详细规格

```
┌─────────────────────────────────┐
│ ┌────┐                          │
│ │照片│  阳光刚好照在街角那棵树上   │  ← 文字摘要：最多20字+...
│ │    │                          │
│ │72px│  📍 南京东路 · 3月20日     │  ← 地名+日期
│ └────┘  🔊 🎬                   │  ← 有环境音/视频时显示小图标
└─────────────────────────────────┘
```

- **位置**：底部弹出，距底部 TabBar 上方 12px
- **尺寸**：宽 362px（屏幕宽 - 40px），高 100px
- **圆角**：16px
- **背景**：白色 98% 不透明度，毛玻璃效果
- **阴影**：`0 -2px 20px rgba(0,0,0,0.07)`
- **弹出动画**：从底部滑入 + 淡入，0.3s，ease-out
- **收起动画**：向下滑出 + 淡出，0.2s，ease-in

**纯文字胶囊（无照片）的预览卡片：**
- 左侧照片位置替换为一个琥珀色渐变的胶囊图标（48x48px，圆角）
- 其余布局不变

### 2.5 "历史上的今天"浮层

**触发逻辑：**
1. 每次进入地图页时查询：`createdAt` 的月日 === 今天的月日，且年份 < 今年
2. 如果有多条匹配，优先展示距今**最久**的那条
3. 同一天只展示一次（用 `lastTodayShownDate` 本地缓存控制）

**交互行为：**
| 操作 | 结果 |
|------|------|
| 自动出现 | 页面加载完成后延迟 0.5s，从顶部淡入下滑（0.4s） |
| 点击卡片 | 进入该胶囊的沉浸式回看页 |
| 不操作等待 5s | 自动向上淡出收起（0.3s） |
| 左滑 | 立即向左淡出收起（0.2s） |

### 2.6 空状态

- 地图正常显示（可缩放拖动）
- 地图中央偏上显示：半透明胶囊图标（48px）+ "你的胶囊会出现在这里"（14px，#9A9A9A）
- **不显示**任何引导按钮、箭头、或指向封存按钮的提示
- 底部 TabBar 正常显示，封存按钮正常可用

### 2.7 定位按钮

- 位置：地图右下角，距底部 TabBar 上方 20px，距右边 20px
- 图标：lucide "locate"，18px
- 容器：40x40px 白色圆形，阴影 `0 2px 8px rgba(0,0,0,0.1)`
- 点击 → 地图动画平移到用户当前位置

## 3. 底部 TabBar

### 3.1 布局规格

```
┌────────────────────────────────────────┐
│  ┌──────────┬──────────┬──────────┐   │
│  │  🗺 地图  │   ＋封存  │ ⏱ 时间线 │   │  ← 浮动胶囊导航栏
│  └──────────┴──────────┴──────────┘   │
└────────────────────────────────────────┘
```

- **外层容器**：宽 100%，高 95px，padding `[12, 21, 21, 21]`
- **胶囊栏**：宽 fill，高 62px，圆角 36px，白色背景 + 1px 边框 #E8E5E0 + 阴影
- **三个 Tab**：等分宽度

### 3.2 Tab 交互

| Tab | 图标 | 文字 | 默认状态 | 激活状态 | 点击行为 |
|-----|------|------|---------|---------|---------|
| 地图 | lucide "map" 18px | "地图" 10px | 图标+文字 #9A9A9A | 琥珀色背景填充，图标+文字白色 | 切换到地图页，保留地图位置和缩放状态 |
| 封存 | lucide "plus" 20px | 无文字 | 40px 琥珀色渐变圆形按钮 | 按下时按钮缩小至 95%（0.1s） | 打开封存页（从底部上滑全屏，0.3s） |
| 时间线 | lucide "timer" 18px | "时间线" 10px | 图标+文字 #9A9A9A | 琥珀色背景填充，图标+文字白色 | 切换到时间线页 |

**封存按钮特殊设计：**
- 中间的封存按钮比两侧 Tab 更突出（40px 渐变圆形 + 发光阴影）
- 没有文字标签，纯图标
- 按下反馈：缩小到 95% + 阴影减弱（0.1s）
- 松开：恢复原大小（0.1s）

### 3.3 页面切换动画

- 地图 ↔ 时间线：无动画，直接切换（标准 TabBar 行为）
- 封存页：从底部全屏上滑进入（0.3s），关闭时下滑退出（0.3s）
- **封存页打开时 TabBar 隐藏**，关闭后 TabBar 恢复

## 4. 封存页

### 4.1 进入与退出

| 触发 | 动画 | 说明 |
|------|------|------|
| 点击封存按钮 | 页面从底部上滑覆盖全屏（0.3s ease-out） | TabBar 同时隐藏 |
| 点击左上角 × | 页面向下滑出（0.3s ease-in），露出地图/时间线 | TabBar 恢复 |
| 右滑手势 | 同上 | 从左边缘 20px 内开始右滑触发 |
| 封存完成 | 封存仪式动效 → 自动关闭封存页 → 跳转地图 | 见 4.10 |

### 4.2 页面加载时序

```
封存页打开（0.3s 上滑动画）
  ↓ 动画完成后
同时触发：
  ├─ 文字输入框自动获取焦点（键盘弹起）
  └─ 定位请求开始
       ├─ 已授权 → 调用 wx.getLocation → 成功 → 调用逆地理编码 → 显示地名
       ├─ 已授权 → 调用 wx.getLocation → 失败/超时(5s) → 显示"定位失败，点击手动选择"
       └─ 未授权 → 显示"点击授权定位" → 点击 → 触发系统授权弹窗
```

### 4.3 定位区域详细交互

```
┌─────────────────────────────────┐
│ 📍 南京东路步行街            ›   │  ← 正常状态
└─────────────────────────────────┘
```

| 状态 | 左侧图标 | 文案 | 右侧 | 点击行为 |
|------|---------|------|------|---------|
| 定位中 | 📍 旋转动画 | "定位中..." | 无 | 不可点击 |
| 定位成功 | 📍 琥珀色 | "{地名}" | › 箭头 | 点击 → 打开位置修正页 |
| 定位失败 | 📍 灰色 | "定位失败，点击手动选择" | › 箭头 | 点击 → 打开地图选点页 |
| 未授权 | 📍 灰色 | "点击授权位置" | › 箭头 | 点击 → 触发 `wx.authorize`；授权后自动开始定位 |
| 授权被拒 | 📍 灰色 | "位置未授权，点击手动选择" | › 箭头 | 点击 → 打开地图选点页 |
| 离线无缓存 | 📍 灰色 | "无法定位，点击标记位置" | › 箭头 | 点击 → 打开地图选点页（离线地图） |
| 手动选择完成 | 📍 琥珀色 | "{手动选择的地名}" | › 箭头 | 点击 → 重新打开地图选点页修改 |

**交互行为：**
| 操作 | 结果 |
|------|------|
| 拖动地图 | 中心标记固定不动，地图移动，松手后反查地名（调用逆地理编码） |

### 4.5 文字输入区

- **Placeholder**："写下此刻的感受..."，颜色 #C8C0B4
- **输入时**：placeholder 消失，文字颜色 #1C1C1C
- **字号**：16px，行间距 1.6
- **高度**：自适应内容，最小占据屏幕中间可用区域
- **键盘**：进入封存页时自动弹出，点击其他区域（媒体按钮等）时键盘收起
- **可为空**：允许纯照片/视频胶囊

### 4.6 媒体按钮栏

```
┌─────────────────────────────────┐
│ [📷 照片]  [📹 视频]  [🎙 环境音] │
└─────────────────────────────────┘
```

三个按钮均为药丸形状（pill shape），水平排列。

#### 4.6.1 照片按钮

| 状态 | 视觉 | 点击行为 |
|------|------|---------|
| 默认（未选照片） | 琥珀色边框 + 琥珀色图标和文字 | 弹出 ActionSheet |
| 已选照片 | 琥珀色填充背景 + 白色图标和文字 + 角标数字 | 弹出 ActionSheet（追加选择） |
| 相机未授权 | 同默认 | 弹出 ActionSheet，但"拍照"选项灰色不可点，仅"从相册选取"可点 |

**ActionSheet 内容：**
```
┌─────────────────────────────────┐
│           拍照                  │  ← 调用 wx.chooseMedia({ sourceType: ['camera'] })
├─────────────────────────────────┤
│        从相册选取                │  ← 调用 wx.chooseMedia({ sourceType: ['album'] })
├─────────────────────────────────┤
│           取消                  │  ← 关闭 ActionSheet
└─────────────────────────────────┘
```

**拍照流程（选择"拍照"后）：**
1. 调用 `wx.chooseMedia({ count: 9 - 已选数量, mediaType: ['image'], sourceType: ['camera'] })`
2. 系统相机打开 → 用户拍照 → 确认

### 4.9 封存流程（点击封存按钮后）

```
点击"封存"按钮
  ↓
按钮变为"封存中" + loading
  ↓
1. 照片压缩（如果有照片）
   └─ 压缩到 1080p + quality 0.8，目标 ≤1MB/张
  ↓
2. 并行上传媒体文件到云存储
   ├─ 照片 × N
   ├─ 视频（如有）
   └─ 环境音（如有）
  ↓
3. 写入云数据库（胶囊记录）
  ↓
4. 上传成功 → 播放封存仪式动效
   OR
   上传失败/无网络 → 保存到本地，toast "已封存到本地，联网后自动同步"
  ↓
5. 封存仪式动效播放完毕 → 关闭封存页 → 跳转地图
```

### 4.10 封存仪式动效（详细描述）

```
阶段 1（0-0.3s）：收缩
  - 封存按钮文字从"封存中"变为 ✓
  - 文字输入区和照片区域开始缩小+淡出
  - 所有内容向屏幕中心汇聚

阶段 2（0.3-0.8s）：成形
  - 汇聚的内容变成一个琥珀色小圆点（胶囊形态）
  - 圆点位于屏幕中央

阶段 3（0.8-1.3s）：落入
  - 小圆点向下飘移，飘向底部 TabBar 地图图标的位置
  - 同时封存页背景淡出
  - 小圆点到达目标位置后，有一个轻微的弹跳效果

阶段 4（1.3-1.5s）：完成
  - 封存页完全消失
  - 地图页显示，新胶囊 Marker 从小到大弹入（0.2s）
  - Marker 有一个短暂的发光脉冲（1次，0.5s）
```

**调性**：像把一颗小石子轻轻投入湖面，看到涟漪扩散。不是烟花庆祝。

## 9. 异常处理详细方案

### 9.1 网络相关

| 场景 | 用户感知 | 技术处理 |
|------|---------|---------|
| 封存时无网络 | Toast："已封存到本地，联网后自动同步" | 数据写入本地缓存，媒体文件保存在临时目录 |
| 恢复网络后同步 | 无感知（后台自动） | 监听 `wx.onNetworkStatusChange`，自动上传 |
| 同步失败 | Marker/卡片显示"同步失败"标识 | 自动重试 3 次（间隔 5s/15s/30s），之后标记为 failed |
| 加载胶囊列表失败 | 地图：使用本地缓存数据；时间线：显示"加载失败，点击重试" | 优先使用本地缓存，降级显示 |

### 9.2 权限相关

| 场景 | 处理方式 |
|------|---------|
| 定位权限首次被拒 | 封存页定位区域显示"位置未授权，点击手动选择" |
| 定位权限二次被拒（系统不再弹窗） | 点击"授权"时直接跳转到 `wx.openSetting()` |
| 相机权限被拒 | ActionSheet 中"拍照"选项不可点击，仅"从相册选取"可用 |
| 麦克风权限被拒 | 视频录制和环境音录制按钮隐藏 |
| 所有权限都被拒 | 仍可使用：手动选位置 + 从相册选图 + 纯文字封存 |

### 9.3 存储相关

| 场景 | 处理方式 |
|------|---------|
| 云存储空间不足 | 封存时弹窗："存储空间不足，暂时无法封存" |
| 本地缓存空间不足 | Toast："设备存储空间不足" |
| 照片压缩失败 | 使用原图上传（不阻断流程） |

### 9.4 内容相关

| 场景 | 处理方式 |
|------|---------|
| 文字和照片都为空时点击封存 | 按钮为不可用状态（灰色），不可点击 |
| 封存中途小程序被杀 | 已输入内容丢失（v1.0 不做草稿保存） |
| 逆地理编码失败 | 显示经纬度坐标："31.24°N, 121.50°E"，不阻断流程 |
| 照片加载失败（回看时） | 显示灰色占位图 + 重试图标 |
| 视频播放失败（回看时） | 显示第一帧静态图 + "视频加载失败" |
| 环境音播放失败（回看时） | 不显示音频波形，不影响其他内容展示 |
