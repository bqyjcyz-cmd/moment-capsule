# Moment Capsule — Product Roadmap

## 版本全景

```
v0.1 最小闭环    v1.0 小程序MVP     v1.1 小程序增强     v2.0 独立App
──────────── → ──────────────── → ──────────────── → ────────────────
"能封存能回看"   "体验完整"         "更有氛围"         "重逢+完整系统"
   2 周            +2-3 周            +2-3 周            8-12 周
```

---

## v0.1 — 可用原型（2 周）

跑通核心闭环：封存 → 地图 → 沉浸式回看

- 微信登录 + 云开发
- 定位 + 地点名称反查
- 文字 + 拍照 + 封存
- 地图视图（胶囊散落）
- 沉浸式回看（基础版）

**里程碑：开始自用，验证核心假设**

---

## v1.0 — 微信小程序 MVP（v0.1 后 +2-3 周）

在 v0.1 基础上补齐完整体验：

- 照片压缩 + 短视频 + 环境音
- 封存仪式动效
- 时间线视图 + 时间筛选
- 沉浸式回看增强（视频/音频/多图）
- 离线封存 + 自动同步
- 编辑 / 删除
- 首次引导 + 权限降级
- "历史上的今天"（轻量被动重逢）
- 空状态设计

**里程碑：体验完整，可日常使用**

---

## v1.1 — 小程序增强版（v1.0 后 +2-3 周）

- 自动天气采集
- 沉浸式回看 v2（天气展示、时间色调）
- 地图聚合展示
- 地图足迹路径
- 胶囊统计概览

**里程碑：封存更丰富，回看更有氛围**

---

## v2.0 — 独立 App（8-12 周）

- 自建后端 + 小程序数据迁移
- 地理围栏重逢提醒 + 智能提醒策略
- 沉浸式回看（完整版，真正全屏）
- 原生 Live Photo 支持
- 数据导出 + iCloud 备份
- 桌面小组件
- 胶囊时间锁
- 年度回顾
- 关键词搜索

**里程碑：重逢体验 + 完整的情感记忆系统**

---

## v3.0+ — 长期愿景

- AI 感受洞察
- 跨设备同步
- 选择性分享
- 声音识别
- 协作胶囊
