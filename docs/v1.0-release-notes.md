## 一、版本概述

此刻胶囊 v1.0 是产品的首个正式发布版本，代号 "First Drop"。实现了从封存→回看的完整闭环，覆盖 Brief 中定义的全部 12 项功能。

**核心价值：在有感触的时刻封存此刻，在未来的某天重新遇见。**

## 四、技术架构

### 前端
- 微信小程序原生开发（无框架）
- 自定义 TabBar 组件
- Lucide 官方 SVG 图标（26 个，按用途分色）
- 异步 Storage API（wx.getStorage/setStorage）

### 后端
- 微信云开发（云函数 + 云数据库 + 云存储）
- 云函数 `capsule`：login / createCapsule / getCapsules / getCapsuleDetail / updateCapsule / deleteCapsule / getTodayInHistory
- 云数据库集合：`capsules`
- 数据流：本地优先写入 → 异步上传媒体到云存储 → 调用云函数写入数据库

### 工具层
- `storage.js` — 本地 CRUD + 云端同步
- `upload.js` — 照片压缩 + 批量上传（照片/视频/音频）
- `location.js` — 权限三态检查（authorized/denied/not-asked）
- `date.js` — 日期格式化（占位符替换，兼容旧 Android）

## 六、已知限制与折中

| 项目 | Brief 预期 | 实际实现 | 原因 |
|------|-----------|---------|------|
| 位置获取 | wx.getLocation 自动获取 | wx.chooseLocation 用户确认 | getLocation 权限审核不通过 |
| 回看动效 | 文字淡入、手写效果 | 无动画，直接展示 | Brief 允许"不追求完美动效" |
| 视频录制 | 自定义录制 UI（按住录制） | 系统 chooseMedia | 功能可用，体验略粗糙 |
| 天气信息 | 自动捕获天气 | 未实现 | 需要第三方天气 API |
