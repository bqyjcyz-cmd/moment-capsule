# 此刻胶囊 v1.0 — 测试用例

> 编写日期：2026-03-30
> 测试范围：全部 v1.0 功能
> 测试环境：iOS 真机 + Android 真机 + 微信开发者工具模拟器

---

## 测试约定

- **P** = 优先级（P0 必测 / P1 重要 / P2 一般）
- **前置条件** = 执行测试前需要的状态
- 每条用例标注 [ ] 供勾选

---

## 一、首次引导（Onboarding）

### TC-1.1 引导页展示与翻页
- **P0** | 前置条件：首次安装或清除缓存后打开
- [ ] 打开小程序，自动跳转到引导页
- [ ] 第 1 页显示"此刻胶囊"标题 + 钻石图标 + 两行说明文字
- [ ] 点击"继续"按钮，滑到第 2 页
- [ ] 第 2 页显示"这不是日记" + 日历图标
- [ ] 点击"继续"，滑到第 3 页
- [ ] 第 3 页显示权限卡片（位置/相机/麦克风）
- [ ] 左右手势滑动可切换页面
- [ ] 底部圆点指示器正确高亮当前页

### TC-1.2 权限授权流程
- **P0** | 前置条件：引导页第 3 页
- [ ] 点击"位置权限"的"授权"按钮 → 弹出系统授权弹窗
- [ ] 授权成功 → 按钮变为"已授权"（绿色）
- [ ] 拒绝授权 → 按钮变为"去设置"
- [ ] 相机和麦克风权限同上逻辑
- [ ] 点击"进入此刻胶囊" → 跳转到地图首页
- [ ] 点击"暂时跳过" → 同样跳转到地图首页
- [ ] 再次打开小程序 → 不再显示引导页

### TC-1.3 引导页边界情况
- **P2** | 前置条件：引导页
- [ ] 快速连续点击"继续"不会跳过页面或崩溃
- [ ] 在第 1 页左滑无反应（不能滑到负数页）

- [ ] 点击地图空白区域 → 预览卡片消失

### TC-2.3 "历史上的今天"
- **P1** | 前置条件：至少有一颗胶囊的创建日期与今天同月同日但不同年

- [ ] 5 秒后卡片自动消失

### TC-2.4 标记点与胶囊映射

- [ ] 无位置信息的胶囊不在地图上显示标记
- [ ] 点击标记 A 显示的预览内容对应胶囊 A（不错位）
- [ ] 有 5+ 颗胶囊时，每个标记点击都能正确对应

## 三、封存页（创建胶囊）

### TC-3.1 页面进入与退出
- **P0** | 前置条件：在地图或时间线页
- [ ] 点击 TabBar 中间 ⊕ 按钮 → 进入封存页
- [ ] 顶部显示关闭按钮 × + 当前时间
- [ ] 点击 × → 无内容时直接返回
- [ ] 点击 × → 有内容时弹出"放弃此次封存？"确认弹窗
- [ ] 确认放弃 → 返回上一页；取消 → 留在封存页

### TC-3.2 地址选择
- **P0** | 前置条件：封存页加载
- [ ] 地址栏显示"正在定位..."或"点击授权定位"
- [ ] 位置权限已授权时 → 自动弹出 `chooseLocation`
- [ ] 用户选择地点 → 地址栏显示选择的地名
- [ ] 用户取消选择 → 地址栏显示"点击重新定位"
- [ ] 点击地址栏 → 重新打开地点选择器
- [ ] 位置权限被拒 → 显示"点击授权定位"，点击后跳转系统设置

### TC-3.3 心情选择
- **P0** | 前置条件：封存页
- [ ] 默认显示"😊 开心"心情
- [ ] 点击心情区域 → 底部弹出心情选择面板（8 种心情）
- [ ] 选择"难过" → 面板关闭，心情区域更新为"😢 难过"
- [ ] 心情面板外部点击 → 面板关闭

### TC-3.4 文本输入
- **P0** | 前置条件：封存页
- [ ] 文本块显示"文本"标签
- [ ] 点击输入区 → 键盘弹出，光标定位
- [ ] 输入多行文字 → 输入区自动增高
- [ ] placeholder 显示"写下此刻的感受..."（灰色斜体）
- [ ] 输入文字后 → 封存按钮从灰色变为绿色

### TC-3.5 照片添加
- **P0** | 前置条件：封存页
- [ ] 初始状态不显示照片块（工具栏相机按钮可点击）
- [ ] 点击工具栏相机按钮 → 弹出"拍摄照片/从相册选择"
- [ ] 选择"从相册选择" → 打开相册，选择图片后照片块出现
- [ ] 选择"拍摄照片" → 相机权限未授权时弹出授权请求
- [ ] 照片块显示"照片 1/8"标签
- [ ] 继续添加照片 → 计数更新（2/8, 3/8...）
- [ ] 达到 8 张 → 添加按钮消失，Toast 提示"最多8张照片"
- [ ] 点击照片右上角 × → 删除该照片，计数更新
- [ ] 照片排列为 3 列网格，正方形缩略图

### TC-3.6 录音功能
- **P0** | 前置条件：封存页
- [ ] 点击工具栏麦克风按钮 → 跳转到录音页
- [ ] 录音页显示"录音"标题 + "取消"按钮
- [ ] 点击中间红色录制按钮 → 开始录音
- [ ] 录音中：计时器走动 + 波形动画 + "正在录音..."提示
- [ ] 点击停止按钮 → 录音停止，显示确认区域
- [ ] 确认区域显示：录音预览卡 + "重新录制" + "确认使用"
- [ ] 点击"确认使用" → 返回封存页，录音块出现（♪ 图标+时长）
- [ ] 点击"重新录制" → 清除录音，回到初始状态
- [ ] 封存页录音块 → 点击 ▶ 可播放，点击"删除"可移除
- [ ] 录音最长 60 秒，到时自动停止

### TC-3.7 视频添加
- **P1** | 前置条件：封存页
- [ ] 可通过 chooseMedia 添加视频（≤15秒）
- [ ] 视频块显示缩略图 + 时长 + 删除按钮
- [ ] 点击视频缩略图可预览播放

### TC-3.8 颜色标记
- **P2** | 前置条件：封存页
- [ ] 工具栏下方显示 9 个颜色圆点
- [ ] 默认选中绿色（#8BA676），有深色边框
- [ ] 点击其他颜色 → 边框切换到选中的颜色
- [ ] 颜色信息随胶囊一起保存

### TC-3.9 封存操作
- **P0** | 前置条件：封存页，已输入内容
- [ ] 有文字/照片/视频/录音中任一 → 封存按钮变绿可点击
- [ ] 无任何内容 → 封存按钮灰色不可点击
- [ ] 点击封存 → 跳转到封存仪式页
- [ ] 封存仪式：便签收缩动画 → 胶囊出现 → 飘走 → 自动返回地图页
- [ ] 返回地图后，新胶囊的标记点出现在地图上

### TC-3.10 封存数据完整性
- **P0** | 前置条件：创建一颗完整胶囊（文字+照片+录音+位置+心情）
- [ ] 在时间线中能看到该胶囊
- [ ] 在地图上能看到标记点
- [ ] 回看时所有信息完整（文字/照片/地点/时间/心情/录音）
