## Dependencies & Implementation Order

```
Phase 1: 基础设施（约 1 周）
├── #6 微信登录 + 云开发存储（数据模型、云函数、存储桶）
├── #2 地理位置自动捕获
└── #12 照片压缩上传

Phase 2: 核心记录（约 1-2 周）
├── #1 胶囊记录（文字+照片）  ← 依赖 Phase 1
├── #8 短视频录制             ← 依赖 Phase 1
├── #9 环境音录制             ← 依赖 Phase 1
└── #7 离线记录 + 自动同步    ← 依赖 #1

Phase 3: 核心回看（约 1-2 周）
├── #3 地图视图               ← 依赖 Phase 2 有数据
├── #5 时间线视图 + 时间筛选  ← 依赖 Phase 2 有数据
└── #4 沉浸式回看             ← 依赖 #3 或 #5 进入

Phase 4: 完善（约 0.5-1 周）
├── #10 胶囊编辑和删除
└── #11 首次使用引导 + 分步授权
```

**预估总周期：4-6 周**

## Parallel Opportunities

- Phase 1 的三个任务可以并行
- #8 短视频和 #9 环境音可以并行开发
- #3 地图视图和 #5 时间线视图可以并行开发
